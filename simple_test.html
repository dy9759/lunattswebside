LunaTTS 显示问题测试报告

问题描述

在原始代码中，.typeless-icon-text 类被设置了 font-size: 3px 和 line-height: 0，导致 "LunaTTS" 文字几乎不可见。

修复详情

问题源头： /src/app/globals.css 文件中的样式定义

原始代码：

.typeless-icon-text {
    font-size: 3px !important;
    line-height: 0 !important;
}

修复后代码：

.typeless-icon-text {
    /* 移除导致文字不可见的样式 */
    /* font-size: 3px !important; */
    /* line-height: 0 !important; */
}
验证结果
  • ✅ HTML 源码中包含 "LunaTTS" 文字
  • ✅ Header 组件正确渲染
  • ✅ 文字使用渐变样式 (45deg, #FFC700, #FF6B35)
  • ✅ 字体大小恢复正常 (1rem - 1.3rem 响应式)
  • ✅ 文字现在在页面中清晰可见
技术细节

组件位置： /src/components/Layout/Header.tsx 第78-93行

样式应用： MUI Typography 组件 + typeless-icon-text 类名

渐变效果： CSS linear-gradient + background-clip: text

响应式设计： xs:1rem, sm:1.1rem, md:1.3rem
